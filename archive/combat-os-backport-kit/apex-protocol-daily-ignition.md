# APEX PROTOCOL
## Daily Ignition — Splash Card Content

---

## USAGE NOTES FOR APP AGENT

- Each entry is one splash card displayed on app open
- Max 3 lines per card. Max ~150 characters per entry.
- Display order: sequential or random — app logic decides
- User can dismiss manually or card auto-fades after 5 seconds
- User can bookmark any card — bookmarked cards accessible in saved section
- Daily ignition toggle: on/off in app settings (default: on)
- No category labels displayed. No week correlation. Quote only.
- Language: English
