# Legacy Spreadsheet System (v1/v1.5)

This directory contains the superseded spreadsheet-based HUD system:
- **fighters-os.gs**: Pure Apps Script HUD (v1)
- **fighters-os-lite.gs**: Lite Apps Script variant (v1.5)
- **build_sheet.py**: Python script for styled sheet generation
- **fighters-os.xlsx**: Generated styled spreadsheet

These files have been superseded by the React PWA + scripts/webhook.gs v2 system.

Archived 2026-07-10.
